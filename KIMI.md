# TTS -- Kimi CLI shim

Canonical routing lives in `./AGENTS.md`. Read that first.

Kimi CLI does not auto-discover project rules; the user `@`-includes at session start. Either `@AGENTS.md` (canonical) or `@KIMI.md` (this file, which routes you to AGENTS.md) works. Existing handoffs that say `@KIMI.md` remain valid.
